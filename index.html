Guia Rápido para Anotações de Veículos no CVAT

Escolha o idioma:

1) Criação de Conta

Para começar, você precisará criar uma conta, caso ainda não possua uma criada.


Se você já possui uma conta, pode simplesmente fazer o login.
Senão, clique em 'Create an account'.

Preencha todos os dados da tela e clique em Submit.

2) Seleção de Jobs

Primeiro, confirme se o JOB desejado está no estágio 'annotation' e se foi atribuído a você. Após confirmar, clique nele para iniciar o trabalho.

3) Visão Geral da Interface do CVAT

Essa imagem mostra os principais paineis de acesso aos recurso do programa.

  • Barra de Ferramentas: Este painel reúne as ferramentas essenciais que usaremos, facilitando o acesso rápido para otimizar o fluxo de trabalho.
    • 1. Cursor,
    • 2. Mover a imagem na tela,
    • 3. Undo / Redo,
    • 4. Restaura o Zoom para Exibir a Imagem Completa,
    • 5. Zoom e
    • 8. Retângulo,
  • Gravar: salva o trabalho no servidor.
  • Barra de deslocamento: contém botões de navegação:
    • 1. Vai para o primeiro frame,

    • 3. Anda 1 frame para trás,
    • 4. Play,
    • 5. Anda 1 frame para frente,
    • 6. Anda 10 frames para frente,
    • 7. Vai para o último frame
  • Produção: Resumo do JOB com todos os objetos anotados.
  • Barra de objetos: Mostra todos os objetos da imagem selecionada.

4) Ferramenta de Anotação a ser utilizada (retângulo)

Depois de escolher um JOB, você vai utilizar a barra de ferramentas lateral para escolher o Rótulo para anotação do objeto desejado.

Nesse projeto trabalharemos apenas com retângulos. Escolha o Rótulo desejado.

4.2) Escolha da Ferramente de Anotação

  1. Escolha a ferramenta de anotação Retângulo.
  2. Escolha o rótulo adequado para a anotação (ex: Carro, Moto, Caminhão).
  3. Selecione a opção By 2 Points.
  4. Clique no botão Track para iniciar o rastreamento.

4.3) Instruções para Realizar a Anotação

Siga estas etapas para fazer uma anotação eficaz:

  1. Desenhe os Retângulos: Clique no canto superior esquerdo do veículo e arraste até o canto inferior direito para formar um retângulo. Repita este passo para as rodas. (Veja a imagem da esquerda para referência).
  2. Avance os Frames: Mova 5 frames à frente. Note que os retângulos ficaram parados no lugar original.
    Mova e ajuste os retângulos na nova posição do veículo e das rodas (imagem da direita).Dependendo do cenário, avançar mais do que isso pode não ser eficaz.
  3. Volte e Ajuste: Retorne aos 5 frames anteriores e ajuste os retângulos para que se alinhem com a nova posição do veículo e das rodas.

4.4) Como finalizar o tracking (IMPORTANTE)

Quando o objeto sair pela metade da cena, acione o botão que o torna invisível. Atenção: Antes de realizar essa ação, assegure-se de que o recurso de zoom esteja desativado para confirmar que o objeto está realmente saindo da cena.

Ao clicar no botão FINALIZAR, o Objeto 5, que está saindo da cena, deve ficar invisível mas os anteriores premanecerão visíveis

Aviso Importante: Nunca utilize a tecla DEL para excluir um objeto, pois essa ação removerá o objeto de todos os quadros anteriores.

5) Como finalizar o JOB (IMPORTANTE)

Para concluir o trabalho com sucesso, é essencial que você o Salve e Finalize a sua tarefa no sistema.

5.1) Como salvar o seu trabalho

Certifique-se de salvar seu trabalho ao concluí-lo, clicando no ícone do disquete localizado no canto superior esquerdo da tela. Recomendamos que você também salve progressivamente durante o desenvolvimento do projeto.

5.1) Como finalizar o JOB

Antes de finalizar o JOB, certifique-se de revisar todas as imagens associadas ao seu trabalho. Após a verificação, acesse o menu localizado no canto superior esquerdo da tela e selecione a opção "Finish the JOB".

Após finalizar o seu JOB, fique atento à confirmação de recompensa.  

Caso não a receba, isso sugere que há erros de anotação em seu trabalho. O JOB será retornado para você fazer as correções necessárias.
Continue ajustando e reenviando até que a tarefa seja aprovada pelos verificadores.

6) Dicas Rápidas

6.2) Utilize as teclas de atalho para agilizar o trabalho

Atalho Função Principal
F1 Abrir/fechar a lista de atalhos disponíveis
F2 Ir para a página de configurações ou voltar
N Repetir o último procedimento de desenho com os mesmos parâmetros
Ctrl+S Salvar o trabalho
Ctrl+Z Cancelar a última ação relacionada a objetos
Ctrl+Shift+Z ou Ctrl+Y Cancelar a ação desfazer
Botão do Meio do Mouse enquanto precionado Mover o quadro da imagem (por exemplo, enquanto desenha)
Clique Duplo no Mouse Restaura o Zoom para Exibir a Imagem Completa

7) Recompensa e Regra de Cálculo

A recompensa para este trabalho será calculada com base nos seguintes critérios:

             ('Total de Retângulos ajustados manualmente' * 0.0025 USDC)

Exemplo

Suponhamos que o total de Retângulos ajustados manualmente seja 2000.

Aplicando a regra:

            (2000 * 0.0025 USDC) = 5.00 USDC

Portanto, você receberá 5.00 USDC.
